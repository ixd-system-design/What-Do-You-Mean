WHAT DO YOU MEAN?

Words, well defined.

This is a demo page to fetch and display data from the Oxford Dictionaries API via a NodeJS backend. This iteration also uses NodeJS as a relay to satisfy the requirements of Cross Origin Resource Sharing (CORS). NodeJS takes care of both the frontend and the backend via a framework called Express. For the backend API, We use Express to create a search endpoint for relaying requests. For the frontend, We also use Express library to serve up a public folder of static assets.

We use Environment Variables to safely store API Keys on the NodeJS Server.

Created by Harold Sikkema in the context of Systems Design in the Interaction Design program at Sheridan College. Fonts via Monotype. Thanks to Oxford Dictionaries for all the words.
